🎳 Bowling Alley

The Hexagonal Challenge by Alistair Cockburn & J.B. Rainsberger

API ACTIVE

This system proves that AI can handle complex multi-app hexagonal architectures with zero interventions. Built using 4 isolated hexagons connected via bridge adapters.

Endpoints:
  • POST /games - Book a new game
  • POST /games/:id/rolls - Record a roll
  • GET /games/:id/scoreboard - View the current score
